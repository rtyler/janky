= Janky CI

Janky is a simple CI system built in Rust. This is performative coding and not
intended to be a production system you can actually use.



* Two binaries:
  * `janky-server`
    * Listens HTTP
    * Does web shit
    * Interacts with agents
  * `janky-agent`:
    * Run workloads
    * Listen HTTP
    * executes commands
